Tier-3 production-level Agentic AI project.

- FastAPI backend
- RAG knowledge base
- SQLAlchemy database

- Tool calling agents
- Clean architecture design

Goal:
Build a real-world level AI system that demonstrates backend engineering and agentic workflows.
